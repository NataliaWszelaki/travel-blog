This project is a simple static website created as part of a university assignment.
It presents a personal travel blog with multiple pages, including a homepage, gallery, trip description, and a contact form.

The website was built using HTML, CSS (Flexbox and Grid), and basic JavaScript.
It includes responsive design, a navigation menu, image gallery with lightbox effect, and form validation.
